Welcome!

About:

Necessitas is a teaching application written in JavaScripthat works with the Clotho 3.0 JavaScript API to demonstrate Clotho's capabilities.

Where to Start:

If you are completely new to Clotho, we recommend starting with the general use tutorial. If you would like to continue and develop your own application for Clotho, the developer tutorial will show you the basics.
