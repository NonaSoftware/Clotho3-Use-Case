.create()

Creates the specified Clotho object(s)

Input: (JSON object) An object describing an instance(s) of a Clotho Schema

Returns: The ID (or sequential list of IDs if more than one object was specified) of the successfully created object.

Notes:
  • If an existing object ID is used, fields present in the original object but not in the input object are unchanged. If a new field is encoded in the object spec, it will be added to the existing object.
  • If the object does not exist or if an object ID is not provided, the object is then constructed by way of Clotho.create()

.set()

Sets all fields present in the object spec to their associated values for the Clotho object represented by the object ID.

Input: (JSON object) An object spec or list of multiple specs containing field-value pairs to be set. Each object spec must contain the object ID to be modified.

Returns: The ID of the modified/created object.

Notes:
  • If the object spec includes an existing ID, fields present in the original object but not in the input object spec are unchanged. If a new field is encoded in the object spec, it will be added to the existing object.
  • If the object does not exist or if an object ID is not provided, the object is then constructed by way of Clotho.create()

.query()

.query() will query for all Clotho objects matching the specified criteria.

.queryOne() will query for any single Clotho object matching the specified criteria.

Input: (JSON object) An object spec.

Returns: The first object matching the fields provided in the spec.

.destroy()

Destroys the specified Clotho object

Input: (Object selector) A String representing the desired Clotho object's unique ID targeted for deletion.

Returns: The ID of the destroyed object

Notes:
  • Clotho will fail to destory an object if given an ambiguous object selector.
  • An error message is reported on the 'say' channel if Clotho.destroy() fails.

.get()

Gets a specified Clotho object

Input: (Object selector) A String representing the desired Clotho object's unique ID.

Returns: The object(s) specified by the input object selector(s)

Notes:
  • If the input object selector is ambiguous, Clotho will return the first object posted by MongoDB (essentially arbitrarily)
  • Clotho will report an error message on the 'say' channel if an object could not be retrieved.
